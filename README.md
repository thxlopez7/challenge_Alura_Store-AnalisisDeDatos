#Análisis de Desempeño y Estrategia de Venta de Tiendas (AluraStoreLatam)

##Resumen del Proyecto

Este proyecto contiene el análisis exploratorio de datos (EDA) y la síntesis estratégica de cuatro tiendas del Sr. Juan (Tienda 1, Tienda 2, Tienda 3, Tienda 4). El objetivo principal fue evaluar el rendimiento financiero, logístico y de satisfacción del cliente de cada unidad para tomar una decisión informada sobre qué tienda debería ser vendida para maximizar el capital de reinversión.

El análisis se realizó utilizando la librería Pandas y la visualización con Matplotlib dentro de un entorno Jupyter Notebook.

##Archivos Principales

AluraStoreLatam (1).ipynb: El archivo Jupyter Notebook que contiene todo el código de limpieza, exploración, análisis y visualización de datos.

data/: Carpeta (asumida) donde reside el dataset original (CSV o similar).

informe_final_consolidado.md: El informe estratégico final con la recomendación de venta y su justificación.

##Recomendación Estratégica

La recomendación fue vender la Tienda 1 debido a su alta valoración de mercado (dada por su alto ingreso) y a sus ineficiencias críticas (alto costo de envío y bajo rating), lo que permite al Sr. Juan obtener el mayor capital y sanear su portafolio.

##Tecnologías Utilizadas

Python (3.x)

Pandas (Manejo y Análisis de datos)

Matplotlib (Visualización de datos)
